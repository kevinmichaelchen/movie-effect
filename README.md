# movie-effect

[Effect][effect]-based clients for movie/TV APIs.

## Packages

### SDKs

| Package                              | Version                                  | Description          |
| ------------------------------------ | ---------------------------------------- | -------------------- |
| [@movie-effect/tmdb][tmdb]           | [![npm][tmdb-badge]][tmdb-npm]           | TMDB API client      |
| [@movie-effect/omdb][omdb]           | [![npm][omdb-badge]][omdb-npm]           | OMDb API client      |
| [@movie-effect/watchmode][watchmode] | [![npm][watchmode-badge]][watchmode-npm] | Watchmode API client |
| [@movie-effect/core][core]           | [![npm][core-badge]][core-npm]           | Shared utilities     |

### CLIs

| Package                                      | Description   |
| -------------------------------------------- | ------------- |
| [@movie-effect/tmdb-cli][tmdb-cli]           | TMDB CLI      |
| [@movie-effect/omdb-cli][omdb-cli]           | OMDb CLI      |
| [@movie-effect/watchmode-cli][watchmode-cli] | Watchmode CLI |
